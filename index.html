Projects

Homerun Miko

UCSC Games Showcase Page

Homerun Miko is an isometric action puzzle game featuring a Japanese shrine maiden with a knack for baseball. It was developed over 6 months as part of my senior capstone project at UCSC with my 9 other teammates. I was the lead programmer on the team and was in charge of timelining programming tasks and leading department meetings. Within the project, I primarily worked on programming the enemy state machines, character animation, and UI design.









Patchwork Harmony


Patchwork Harmony is a 2D puzzle and narrative focused game that I finished in 2019. It dives into topics of loss and the passing of time through the eyes of a stuffed bunny. Within my team of 4, I was the primary producer, puzzle designer, and narrative lead. During the production process, I also helped to program the level functionality and storyboarded the 3 different levels present in the game as well as the narrative beats.

Portfolio

Menu UI Implementation

  • Main Menu 10/13/20

    Original main menu screen for Homerun Miko prior to any UI art implementation. Here, I used default Unity canvas UI buttons to create the functionality for the menu.

  • Main Menu 6/6/21

    For our 1.0.0 release, I redeveloped the main menu to have more functionality and worked with our UI artist to create sprites for normal, hover, and click for buttons. We also found fonts that fit our game.

  • Main Menu 8/17/21

    In release 1.4.0, I reviewed wireframes with my team to consolidate functionality and declutter the screen to enhance the UX. The continue button contains the level select and isn't an available option at the start.

  • Button Demo 8/17/21

    Each button has a normal, hover, and click state that also plays a programmed UI option open, close, or selection sound.


  • Settings Demo 8/17/21

    The settings menu is themed around baseball as shown in the volume sliders. I implemented the sliders as well as the buttons below that take you back to different parts of the game.



Dialogue Box UI and VFX Design

  • Dialogue Box Demo 8/17/21

    Homerun Miko's dialogue is executed using the Yarn Unity package, but all the assets and character pop-ups are done completely by hand. I programmed the functionality for the dialogue box UI and worked with my team to allow for text skipping without missing out on vital in-dialogue triggers. Part of the UI is also hidden during dialogue (as shown in the top left) as the player most likely wouldn't have to view or reset the level while speaking.

  • Dialogue VFX Demo 8/17/21

    As part of our continued production following our graduation, we worked on giving more life to our dialogue. I expanded our character manager scripting to allow for dynamic changing of character expressions. I also took the static dialogue VFX drawn by our artists and animated them by hand in the Unity engine. I specifically changed the script such that our narrative lead would simply need to add certain keywords to automatically trigger these dialogue cues.
